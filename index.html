1.Introduction

The objective of this project is to determine the distance of underground  cable fault from the base station in km using an Arduino board. Cables have some resistance,which varies with length  Any deviation in resistance value at that point indicates fault point  Which  represents the standard of distance from the base station. This valueis displayed by display unit for this, it is necessary to know where the cable is located and the route It takes. as locating the fault is complex job due to underground plants installed, it requires more knowledge experience too. 
This is section for Background of our project
